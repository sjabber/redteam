템플릿 관리

템플릿 관리에서는 악성 메일 훈련에 사용할 템플릿을 추가, 수정, 삭제할 수 있습니다. 기본 템플릿이 존재하며 해당 템플릿을 이용해 새 템플릿을 추가해 수정할 수 있습니다.

1. 템플릿 추가

템플릿 추가는 기본 템플릿의 추가 버튼(①)을 눌러서 가능합니다

기본 템플릿에서 수정하는 방식으로 추가되며, 직접 템플릿을 수정하고 표기된 형식을 이용해 대상자 별로 추가 정보를 기입할 수 있습니다.


형식 설명
{{target_name}} 훈련 대상자의 이름
{{count_ip}} count할 수 있는 IP
{{target_position}} 직급
{{target_orgarnize}} 조직
{{target_phone}}훈련 대상자의 전화번호

2. 템플릿 수정 및 삭제

템플릿 수정은 연필 모양의 수정 버튼(②)을 눌러 가능하며, 사용 방법은 템플릿 추가와 동일합니다. 기본 템플릿이 아닌 사용자가 추가한 템플릿은 삭제가 가능하며, 수정 버튼을 누른 후 삭제 버튼을 누르면 템플릿이 삭제됩니다.
